# LifeOS – Life Operating System

**LifeOS** is a personal productivity and life management backend built with Spring Boot. The goal is to create a REST API that serves as the operating system for everyday life –
from daily habits and todos to tracking finances, nutrition, and more.

---

## 🚀 About the Project

LifeOS gives you full control over your daily life through a clean, extensible REST API. Instead of juggling multiple apps, LifeOS centralizes everything in one system that you control.

**Currently implemented:**
- ✅ Todo management
- ✅ Habit tracking
- ✅ File upload for profile pictures

**Planned features:**
- 💰 Finance tracking
- 🍎 Nutrition tracking
- 🔐 Authentication & Authorization (JWT)
- 📊 Statistics & Dashboards
- ... and much more
